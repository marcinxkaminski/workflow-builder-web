### Installing

* Project setup
    ```
    npm install
    ```

* Compiles and hot-reloads for development
    ```
    npm run start
    ```

* Compiles and minifies for production
    ```
    npm run build
    ```

* Run your tests
    ```
    npm run test
    ```

* Lints and fixes files
    ```
    npm run lint
    npm run lint:fix
    ```

### Deployment
```
npm run deploy
```
